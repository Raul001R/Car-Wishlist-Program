## What I Learned

- How to use **Object Oriented Programming (OOP)** in Python by creating a `Car` class and a `CarWishlist` class.
- How to use the `__str__` method to control how objects are displayed.
- How to **read and write CSV files** using Python's built in `csv` module with `DictWriter` and `DictReader`.
- How to **persist data** between sessions by saving to and loading from a file.
- How to build a **menu driven loop** using `while True` and `break` in Python.
- How to use `enumerate()` to display numbered lists.
- How to structure a Python project across **multiple files** and import between them.
